User account cards

From the screens provided User account cards fall into the following categories:

  1. From and To cards
  2. Account status cards
  3. Account information cards
  4. Account value cards
  5. Quick links cards

Each card can contain a panel either on the left or right side associated with the names "sidebar-left or sidebar-right" as well as "header and footer"

Each card can contain a user interface brand colour code that can be used as either an outline or a block colour fill see examples
